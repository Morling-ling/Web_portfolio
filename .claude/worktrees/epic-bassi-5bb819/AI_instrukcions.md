# AI Instructions — Portfolio Website

## Project Goal
This is a personal portfolio website for Lena, a communication design student focusing on illustration, concept art, visual storytelling, and experimental AI-assisted design.

The website should present projects in a clean, editorial, Swiss-inspired layout with strong typography and careful use of color.

The site must remain simple and easy to maintain.

## Developer Experience Level

The user is a design student with little to no coding background.

Rules:
- Keep all code simple and easy to understand
- Avoid complex abstractions
- Do not use advanced patterns unless absolutely necessary
- Do not introduce frameworks (React, Vue, etc.)
- Prefer clear HTML + CSS solutions over JavaScript
- Add comments to explain non-obvious code
- Avoid creating unnecessary files or folders

The code should be easy to edit and maintain by a beginner.

## Editing Behavior

- Make small, controlled changes
- Do not rewrite large parts of the project unless requested
- Always preserve working functionality
- Prefer incremental improvements over large changes

# Design System

## Typography

Two Google Fonts are used:

### Body / UI Font
Poppins Light

Use for:
- body text
- descriptions
- navigation
- metadata
- UI elements

### Display / Accent Font
Abril Fatface

Use for:
- headlines
- section titles
- emphasis elements
- large typographic moments

Rules:
- Do not introduce additional fonts
- Keep typography minimal and elegant
- Avoid excessive font sizes or heavy styling

# Visual Direction

The design should follow a minimal, editorial style inspired by Swiss typography.

Key principles:

- strong grid layout
- large negative space
- clear typographic hierarchy
- elegant but restrained visual style
- focus on artwork and projects

Avoid:

- overly decorative UI
- gradients unless subtle
- complex animations
- heavy shadows
