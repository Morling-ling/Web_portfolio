# Herbal Medicine Label

Type: Illustration / Packaging Design  
Year: 2025  

## Description
A label design for a Chinese herbal medicine product, combining illustration and typographic composition.

## Context
Personal / freelance project exploring packaging design and cultural visual language.

## Focus
The project explores plant-based illustration and a calm, balanced composition inspired by traditional herbal medicine aesthetics. The goal was to create a label that feels both informative and atmospheric.

## Modal Description
A personal and freelance packaging project exploring cultural visual language through Chinese herbal medicine.

The label combines plant-based illustration with a calm, balanced composition to feel both informative and atmospheric.

## Tools
Illustrator, Photoshop
